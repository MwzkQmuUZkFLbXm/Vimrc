---

Usage

1. Vundle的配置   
```$ git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim```

2. 下载插件   
```git clone git@github.com:kmlg/Vimrc.git && cd Vimrc```   

3. 替换原始文件   
```mv vimrc ~/.vimrc```

4. vim命令模式下执行  
     
    * 安装插件       
        ```:PluginInstall```

    * 更新插件    
        ```:PluginInstall```
        
     * 卸载插件(先在.vimrc注释掉不用的插件，然后执行)  
        ```:PluginClean```
