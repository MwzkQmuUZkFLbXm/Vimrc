#### 配置

1. Vundle的配置   
```$ git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim```

2. 下载插件   
```git clone https://github.com/kmlg/Vimrc.git && cd Vimrc```   

3. 替换原始文件   
```cp vimrc ~/.vimrc```

---

#### vim命令模式下执行  
     
* 安装插件或更新插件     
<code>:PluginInstall</code>
	
* 卸载插件(先在.vimrc注释掉不用的插件，然后执行)    
<code>:PluginClean</code>
