使用Vundle进行插件管理

Usage
=====

1. Vundle的配置   
```git clone http://github.com/gmarik/vundle.git ~/.vim/bundle/vundle```

2. 下载插件   
```git clone git@github.com:kmlg/Vimrc.git```   

3. 替换原始文件   
```mv vimrc ~/.vimrc```

4. vim命令模式下执行  
     
    * 安装插件       
        ```:PluginInstall```

    * 更新插件    
        ```:PluginInstall```
